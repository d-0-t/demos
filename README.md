## 🚀 How to Download and Play

GitHub can be a little tricky to navigate if you are just looking to download a game.
Here is exactly how to get things running on your machine:

### 1. Downloading a Game
1. Click on the file you want to play from the list above (for example, `Snek2026-1.0.0.jar`).
2. On the next page, look towards the top right of the file box for the **Download raw file** button (it looks like a tray with a downward-pointing arrow). Click it to save the file to your computer.

### 2. Running a Java Game (.jar)
To play any file ending in `.jar`, you will need to have [Java installed](https://www.java.com/en/download/) on your computer.
* **The Easy Way:** Once downloaded, simply double-click the `.jar` file to launch the game.
* **The Terminal Way:** Open your terminal or command prompt, navigate to your downloads folder, and run: `java -jar filename.jar` (replace `filename` with the actual name of the file).
